# `ItemRemake`
    input: str currentBox, str name = "предмет", int amount = 1, bool isEdit = False

    Класс создающий новое диалоговое окно через которое можно добавить новый предмет и его количество или изменить параметры уже существующего

    draw
    input: str name, int amount
    output:
    Фукция отрисовки окна в поле ввода названия вставляет "предмет" по умолчанию или имя коробки, которая будет изменена; в полле ввода количества вставляет 1 по умолчанию или коичество предмета, который будет изменен

    reject
    input:
    output:
    Функция закрытия окна в случае отказа

    getInfo
    input: 
    output: str currentBox, str name, int amount, bool isEdit
    Функция получения названия и количества 

    accept
    input: str currentBox, str name, int amount, bool isEdit
    output:
    Функция подтверждения добавления в БД нового предмета или изменение существующего
